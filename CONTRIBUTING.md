# Contributing to PharmaGuard

## Development Setup

1. Fork the repository
2. Clone your fork
3. Set up backend and frontend as described in README.md
4. Create a feature branch
5. Make your changes
6. Test thoroughly
7. Submit a pull request

## Code Style

- Python: Follow PEP 8
- TypeScript/React: Use ESLint and Prettier
- Commit messages: Use conventional commits

## Testing

- Test with sample VCF files in `test_data/`
- Verify JSON output matches schema
- Test error handling for invalid inputs
